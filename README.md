1. Clone the repository:

```
$ git clone git@https://github.com/gispatial/TripCarteFrontity.git 

```
2. CD to the cloned directory:

```
$ yarn

```

3. After yarn or npm installation:

```
$ yarn start
It will load to http://localhost:3000/
